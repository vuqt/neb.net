# neb.net

neb.net is the Nebulas compatible C# API. 
Users can sign/send transactions and deploy/call smart contract with it.

## Requirements
neb.net requires the following:

- .NET Framework v4.5.2
- Newtonsoft.Json v11.0.2
- RestSharp v106.3.0
